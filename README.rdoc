== FeedAggregator

Rails3 compatible Feed Aggregation engine

=== Install

add to Gemfile
  gem "feed_aggregator", :git => "git://github.com/ithouse/lolita_feed_aggregator.git", :branch => "master"

run the generator
  rails g feed_aggregator

configure the aggregator via config/aggregator.rb
